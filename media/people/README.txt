CAST / CREW PHOTOS
==================
Drop square-ish portrait photos here (jpg/png).

People are defined ONCE in index.html in the `people` registry
(inside the <script>), e.g.:
  'audrey-tautou': { name: 'Audrey Tautou', photo: 'media/people/audrey-tautou.jpg' },

Set that person's `photo` to the file you dropped here. If photo is
left as '' an outline person icon is shown instead.
Each film references people by their key, so the same person is reused
across films (and across cast + crew) without re-typing.
